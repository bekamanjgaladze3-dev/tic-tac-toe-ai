# Tic Tac Toe Against AI

A simple Tic Tac Toe game built with **HTML, CSS, and JavaScript** where the player competes against an AI opponent.

## 🎮 Features

- Play Tic Tac Toe against AI
- Detects winning combinations
- Highlights the winning cells
- Restart game functionality
- Responsive design for different screen sizes

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run the Project

1. Clone the repository

git clone https://github.com/bekamanjgaladze3-dev/tic-tac-toe-ai.git

2. Open the project folder

3. Open `index.html` in your browser
